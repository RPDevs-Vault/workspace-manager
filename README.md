---

## 🏛️ Structure

- **`/adr/`**: Architectural Decision Records documenting system decisions and criteria.
- **`/skills/`**: Custom instruction booklets, templates, and reference materials for agents.
- **`/templates/`**: Baseline Markdown blueprints for tasks, plans, and walkthroughs.
- **`/memory/`**: Distilled Knowledge Items (KIs) describing the organization's rules.

---

## 🧪 Validation & Checks

This repository contains basic linting workflows to ensure that ADRs and templates match the required standard format and markdown structure.
